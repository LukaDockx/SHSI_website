SHSI attendance workflow

Attendance Checker

Upload Jumbula CSV exports locally in your browser, run the matching logic, and print/export the attendance check sheets.

Privacy: CSV data never leaves this browser. Files are stored in localStorage only on this device until you clear them.

1. Upload CSV files

Drag and drop each CSV into its box, or click a box to browse.

Today attendance CSV Current Jumbula live attendance export No file loaded
Yesterday housing CSV Previous evening housing attendance export No file loaded
Registration database CSV Stable roster/contact/housing/program export No file loaded
Faculty contacts CSV Program, faculty, contact person, phone No file loaded
Settings and program lists

The old script used hard-coded housing/program names. You can edit them here for a new year. Lists are saved in this browser.

2. Faculty attendance review

Upload the four CSV files, then press Review faculty submissions.

3. Reports

Attendance check sheets will appear here after faculty review.

Static app for GitHub Pages. No server, no student data in source code.
